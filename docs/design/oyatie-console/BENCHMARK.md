## 구조적 격차 (모듈 공통 — 백엔드 없이는 불가)
- **실시간·멀티유저**: 협업 동시성·프레젠스·알림 push 전부 시뮬레이션.
- **스케일**: 1,284명·수백 행은 시드 — 가상 스크롤·서버 페이지네이션·인덱스 검색 없음.
- **영속성**: 세션 state(일부 localStorage) — 실 DB·이벤트 소싱 없음(HANDOFF 계약으로 문서화).
- **i18n·접근성**: ko 단일·스크린리더 부분(aria 신규 표면 위주).

## 우리가 벤치마크보다 나은 것 (차별)
- **단일 온톨로지 위의 전 모듈**: Workday+Slack+SAP급 표면이 한 그래프·한 감사 체인·한 정책 엔진을 공유 — 벤치마크들은 서로 분리.
- **가드레일 일관성**: 전 액션 fail-closed·four-eyes·egress 게이트 동일 문법.
- **완전 추적성**: 텍스트 개체 코드=실링크, 수치=drill 불변식.
