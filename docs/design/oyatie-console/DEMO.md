# 데모 대본 (from-scratch 시나리오 — 전부 UI no-code · 2026-07-21)
> 각 대본 = 실제 클릭 경로. 모든 단계는 감사 로그에 남는다 (감사 로그 화면에서 역추적 가능).

## 1. 법인 신설 (그룹 내 새 회사 셋업)
조직도 → 편집 → 「+ 새 법인」 → ① 기본: 법인명·약칭·업종 → ② 조직: 최초 사업장(필수)·팀 칩·결재선 프리셋 → ③ 통합: 셋업 채널·알림·주52h 감시 시드 토글 → 「셋업 개시」.
결과: 스코프 세그·조직도 열·그래프 개체(OB-)·「# 약칭 셋업」 메신저 채널·알림·자동화 초안. 확정 = 법인 카드 → 조직 개편 결재(사전점검→SoD→발효).

## 2. 새 규칙 추가 (자동화)
자동화 → 워크플로 스튜디오 → 「새 규칙」 → 트리거 블록(근태 이벤트 등) → 조건 술어(field·op·value) → 액션(알림·기안 생성) → 「시뮬레이션」(현재 데이터 평가) → 초안 저장 → 「활성화」=게시 승인.
개인 규칙: 범위 세그 「개인」 = 즉시 활성(본인 한정).

## 3. 기존 규정 변경 (규제 파라미터·정책)
컴플라이언스 → RG- 행(예: 최저임금 2027) → 개체 카드 → 「개정 시작」(v+1 초안·현행 유지) → 발효일 지정 → 상신 → four-eyes 승인 → 발효.
접근 정책: 권한·정책 → 정책 캔버스(주체·리소스·조건) → 시뮬레이션(대상자 미리보기) → 개정 스테이징 → proposal 큐 승인.

## 4. 팀·법인 간 협업 (메신저·메일·개체·자동화)
근태 결원 발생 → 커버 플래너 → 대근 편성(인력풀 호출 → 건별 계약 C-D 발행 → passkey 수령=투입 확정) → 메신저 채널에 WO-/AP- 코드 입력=실링크 → 결재 승인 → 급여 회차 대근비(SR-206) 자동 반영 → 대시보드 마진 즉시 갱신.
메일: 첨부 → 「인제스트로」/「증거 등재」 → 개체화 → 관련 채널·기안에 코드로 참조.

## 5. 직원 입사 → 급여 (전 생애주기)
채용: 공고 등록 → 지원자 평가·오퍼 → 입사 확정 = 직원 개체 생성. (경력 즉시: 인사 관리 → 직원 등록.)
근태 예외 처리(사유 필수) → 법인 마감 4/4 → 급여 계산 → 예외 5건 확인 → 상신(AP-) → 이체 예약. 본인: 개인 수신함 passkey 수령확인.

## 6. 분석·예측 저작
경영 분석 → 「AN- 저작」 → 근거 개체 선택·산식·임계 → 라이브 미리보기 → 초안 생성(목록·그래프 합류) → 개체 카드에서 상신·활성.
예측: 예측 화면 → 「FC- 예측 저작」 → 모델(EWMA/Poisson/M/M/c)·시리즈·창.

## 7. 개체 타입·인제스트 확장
온톨로지 → 「+ 타입 제안」 → 스키마 빌더(레이어·속성·관계) → 초안 → 매니저 상신·승인.
인제스트 → 「소스 추가」(종류·주기·템플릿) → 연결 대기 → 업로드/폴 → 필드 검증 → 기대치 게이트 → 적재=개체 생성.
