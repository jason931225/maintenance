- [x] **대근(substitution) 워크플로** — 타임라인 결원 행 「대근 편성」 → 인력풀 배정 모달(`subOpen`/`subAssign`) → **대근 승인 AP- 상신** + 대체자 타임라인 fill-in 실시간 병합 + 결원 노트 flip + 급여(일당/시급) 반영 노트 + 알림 + 감사. 검증: 조이슨 무단결근 → 이대체 배정 → 타임라인 갱신·노트 flip.

- [x] **월간 뷰 계획 vs 실적**: 오늘 타임라인처럼 월간 셀도 계획 vs 실적 구분 — **유급휴가(`lv`)·승인 부재(`pa`, 무급/신청 승인) = 계획 부재(승인·차분: purple/purple-tint)** vs **무단결근(`ab`) = 실적 이탈(red)** vs **결근·대근 대체(`abc`) = 결원 커버(red + 커버 마커)**. `subAssignments`(피대체자 이름) 라이브 반영 → 피대체자 7/3 셀 `ab`→`abc` + 툴팁에 대체자 명시, **대근 배지**(개인 "대근 대체" · 노드 "대근 N")가 팀→사업장→법인 상향 전파. 계획 부재(`lv`/`pa`)는 출근율·결근 집계에서 제외(승인), 대근 대체는 여전히 본인 결근으로 계상. 범례 갱신. 검증: 조이슨 7/3 무단결근→이대체(일용직) 대근→`abc`+배지, 이종호 승인 부재(pa), 6/16 조이슨 pa.
